# HOW TO USE
> RUSTのカーゴボットが来たら通知するbotです。
## インストール
```bash
npm install
```
## 認証
アカウント認証コマンド
１個目やってるときはブラウザで認証
２個目やってるときはRUSTでペアリングしないといけない
```bash
npx @liamcottle/rustplus.js fcm-register
npx @liamcottle/rustplus.js fcm-listen
```
